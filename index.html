The ZGLUE SDK (Alpha) 
 ![The Zglue SDK](zglue_logo_mini_color.svg)

Welcome to the Zglue SDK.

What is provided is only one part of a development environment necessary to work with a zip. The goal is to give you enough information and advice that you can have a simple app running by the end of the day (defined as 14 hours after awakening).

This SDK must be combined with the NuttX rtos, the ARM gcc toolchain (what we use in house), a Linux environment (Ubuntu, Fedora). You will need git to efficiently download the NuttX source, and python (2.7) for several scripts supporting some of the pieces. App Note (XXX) describes installing a Zglue SDK on an Ubuntu system.

Modules covered in this document.

  • Programming with the FAST api is covered as well as
  • Examples for accessing the interface.
  • Implementation of th Nim BLE code common to a number of rtos providersr
  • Examples for working with it,
  • The Nuttx support for adding boards with an example provided

Within each module are

  • descriptions of
    • functions
    • data structures
    • variables of the functions
  • Code snippets to clarify use where appropriate.

Helpful pointers to

  • app notes,
  • hardware details for the dev board
  • Discussion of inhouse experience with the feature.

In the future please look to new versions as more data and information are added to cover the develpment environment in one stop.
